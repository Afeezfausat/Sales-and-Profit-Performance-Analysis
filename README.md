This project analyzed sales and profit performance across products, customer segments, and geographic regions to identify key drivers of revenue and profitability. The goal of the analysis is to help businesses understand which product categories generate the most profit, how customer segments contribute to revenue, and how discounting impacts profitability.
Using Power Query for data preparation and Power BI for visualization, an interactive dashboard was created to provide insights into revenue trends, profit margins, customer behavior, and product performance. The analysis enables businesses to make data-driven decisions about pricing strategies, product focus, customer targeting, and regional expansion.
